## 默认设置

- 行尾为 `lf`（`\n`）
- `font-size: 18`

## 1. 多行同时编辑

同时编辑多行可以使用快捷键 `Shift + Alt + 鼠标左键`

## 2. 注释

多行注释：`Shift + Alt + A`

## 3. 格式化

`Shift + Alt + F`

## 4. 设置 lf

设置里搜索 `eol` ，设置默认行尾字符为 `\n`
