# 拖拽表格行实现移动表格行

可以用原生 HTML5 拖拽事件来做。核心思路是：

1. 给每一行 `tr` 设置 `draggable="true"`。
2. 监听开始拖拽事件（`dragstart`），记录当前被拖拽的行，给它添加一个 `dragging` class。
3. 监听拖拽元素在当前元素上移动事件（`dragover`），清除所有行的 `drop-above` 和 `drop-below` class，通过比较鼠标位置和当前行的边界，判断是拖拽到上半部分还是下半部分，给当前行添加 `drop-above` 或 `drop-below` class 来显示放置提示。
4. 监听拖拽元素在当前元素上放置事件（`drop`），判断放置元素在上半部分还是下半部分来决定将拖拽行插入到当前行的前面还是后面。
5. 监听结束拖拽事件（`dragend`），清除 `dragging` class，重置记录的拖拽行，清除所有行的 `drop-above` 和 `drop-below` class，刷新序号显示。

> 在目标行的 `dragover` 里 `preventDefault()`，允许放置。

css 设计思路：

1. tr 的样式设置 `cursor: grab`。
2. tr 拖拽时设置 `dragging` class，样式设置 `opacity: 0.45`。
   - 此时设置 cursor 会不生效，因为拖动时浏览器会自动使用默认的拖动光标。
3. 当有 tr 拖拽到当前 tr 的上方，给当前 tr 添加 `drop-above` class，样式设置 `box-shadow: inset 0 2px 0 #2563eb`，在当前 tr 的上边缘显示一条蓝色线，提示用户放置位置。
4. 当有 tr 拖拽到当前 tr 的下方，给当前 tr 添加 `drop-below` class，样式设置 `box-shadow: inset 0 -2px 0 #2563eb`，在当前 tr 的下边缘显示一条蓝色线，提示用户放置位置。

下面是一个可直接运行的完整例子：

```html
<!doctype html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>原生表格行拖拽排序</title>
  <style>
    .wrap {
      max-width: 720px;
      margin: 0 auto;
      background: #fff;
      border: 1px solid #e5e7eb;
      border-radius: 10px;
      box-shadow: 0 6px 20px rgba(0, 0, 0, 0.06);
      overflow: hidden;
    }

    table {
      width: 100%;
      border-collapse: collapse;
    }

    th,
    td {
      padding: 12px 14px;
      border-bottom: 1px solid #eef2f7;
      text-align: left;
      font-size: 14px;
    }

    th {
      background: #f8fafc;
      color: #334155;
    }

    tbody tr {
      cursor: grab;
      user-select: none;
      background: #fff;
      transition: background 0.15s ease;
    }

    tbody tr:hover {
      background: #f8fbff;
    }

    tbody tr.dragging {
      opacity: 0.45;
    }

    tbody tr.drop-above {
      box-shadow: inset 0 2px 0 #2563eb;
    }

    tbody tr.drop-below {
      box-shadow: inset 0 -2px 0 #2563eb;
    }
  </style>
</head>
<body>
  <div class="wrap">
    <table>
      <thead>
        <tr>
          <th style="width: 80px">序号</th>
          <th>任务</th>
          <th style="width: 140px">负责人</th>
        </tr>
      </thead>
      <tbody id="tableBody">
        <tr draggable="true"><td>1</td><td>需求评审</td><td>Kevin</td></tr>
        <tr draggable="true"><td>2</td><td>接口联调</td><td>Alice</td></tr>
        <tr draggable="true"><td>3</td><td>前端开发</td><td>Bob</td></tr>
        <tr draggable="true"><td>4</td><td>测试修复</td><td>Chris</td></tr>
        <tr draggable="true"><td>5</td><td>上线发布</td><td>Doris</td></tr>
      </tbody>
    </table>
  </div>

  <script>
    const tbody = document.getElementById("tableBody");
    let draggingRow = null;

    function clearDropClass() {
      [...tbody.rows].forEach((row) => {
        row.classList.remove("drop-above", "drop-below");
      });
    }

    function refreshIndex() {
      [...tbody.rows].forEach((row, idx) => {
        row.cells[0].textContent = idx + 1;
      });
    }

    [...tbody.rows].forEach((row) => {
      row.addEventListener("dragstart", () => {
        draggingRow = row;
        row.classList.add("dragging");
      });

      row.addEventListener("dragend", () => {
        row.classList.remove("dragging");
        draggingRow = null;
        clearDropClass();
        refreshIndex();
      });

      row.addEventListener("dragover", (e) => {
        if (!draggingRow || draggingRow === row) return;
        e.preventDefault();

        clearDropClass();

        const rect = row.getBoundingClientRect();
        const isAbove = e.clientY < rect.top + rect.height / 2;
        row.classList.add(isAbove ? "drop-above" : "drop-below");
      });

      row.addEventListener("drop", (e) => {
        e.preventDefault();
        if (!draggingRow || draggingRow === row) return;
        e.preventDefault();

        const rect = row.getBoundingClientRect();
        const isAbove = e.clientY < rect.top + rect.height / 2;

        if (isAbove) {
          tbody.insertBefore(draggingRow, row);
        } else {
          tbody.insertBefore(draggingRow, row.nextSibling);
        }
      });
    });
  </script>
</body>
</html>
```
