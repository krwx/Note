聚合函数是对一组值执行计算，并返回单个值的函数。

聚合函数经常与 `SELECT` 语句的 `GROUP BY` 子句一起使用。`GROUP BY` 子句将结果集分割成多组值，聚合函数可以用于为每组返回单个值。

最常用的 SQL 聚合函数是：

- `MIN()` - 返回选定列中的最小值
- `MAX()` - 返回选定列中的最大值
- `COUNT()` - 返回集合中的行数
- `SUM()` - 返回数值列的总计
- `AVG()` - 返回数值列的平均值

## MIN()

`MIN()` 函数返回所选列中的最小值。

`MIN()` 函数适用于数值、字符串和日期数据类型。

语法

```sql
SELECT MIN(column_name)
FROM table_name
WHERE condition;
```

示例

返回 Products 表中 Price 列的最低价格：

```sql
SELECT MIN(Price)
FROM Products;
```

返回 Employees 表中 BirthDate 列的最早的出生日期：

```sql
SELECT MIN(BirthDate) AS EarliestBirthdate
FROM Employees;
```

与 ```GROUP BY``` 一起使用。返回 Products 表中每个类别的最低价格：

```sql
SELECT MIN(Price) AS SmallestPrice, CategoryID
FROM Products
GROUP BY CategoryID;
```

### 设置列名（别名）

当您使用 MIN() 时，返回的列将没有描述性名称。

要为列提供描述性名称，请使用 `AS` 关键字：

实例

```sql
SELECT MIN(Price) AS SmallestPrice
FROM Products;
```

## MAX()

MAX() 函数返回所选列的最大值。

MAX() 函数适用于数值、字符串和日期数据类型。

语法

```sql
SELECT MAX(column_name)
FROM table_name
WHERE condition;
```

示例

返回 Products 表中 Price 列的最高价格：

```sql
SELECT MAX(Price)
FROM Products;
```

## COUNT()

COUNT() 函数返回匹配指定条件的行数。

COUNT() 语法

```sql
SELECT COUNT([DISTINCT] column_name | *)
FROM table_name
WHERE condition;
```

COUNT() 的行为取决于括号内使用的参数：

- `COUNT(*)` - 计算表中的总行数（包括 NULL 值）。
- `COUNT(column_name)` - 计算列中所有非 NULL 值的数量。
- `COUNT(DISTINCT column_name)` - 仅计算列中唯一的、非 NULL 值的数量。

**实例**

返回 Products 表中的总行数。

```sql
SELECT COUNT(*)
FROM Products;
```

计算 ProductName 列中所有非 NULL 值的数量

```sql
SELECT COUNT(ProductName)
FROM Products;
```

获取 Products 表中有多少种不同的价格

```sql
SELECT COUNT(DISTINCT Price)
FROM Products;
```

## SUM()

SUM() 函数用于计算指定数值列中值的总和。 SUM() 函数会忽略列中的 NULL 值。

SUM() 语法

```sql
SELECT SUM(column_name)
FROM table_name
WHERE condition;
```

**实例**

返回 OrderDetails 表中所有 Quantity 字段的总和：

```sql
SELECT SUM(Quantity)
FROM OrderDetails;
```

### 带有表达式的 SUM()

SUM() 函数内的参数也可以是一个表达式。

如果我们假设 OrderDetails 列中的每个产品成本为 10 美元，则可以通过将每个数量乘以 10 来计算出总收益（美元）：

```sql
SELECT SUM(Quantity * 10)
FROM OrderDetails;
```

我们也可以将 OrderDetails 表与 Products 表连接起来，以查找实际金额，而不是假设它为 10 美元：

```sql
SELECT SUM(Price * Quantity)
FROM OrderDetails
LEFT JOIN Products ON OrderDetails.ProductID = Products.ProductID;
```

## AVG()

AVG() 函数返回数值列的平均值。

AVG() 函数会忽略列中的 NULL 值。

语法

```sql
SELECT AVG(column_name)
FROM table_name
WHERE condition;
```

**实例**

查找所有产品的平均价格：

```sql
SELECT AVG(Price)
FROM Products;
```

列出所有价格高于平均值的记录，在子查询中使用 `AVG()` 函数：

```sql
SELECT * FROM Products
WHERE price > (SELECT AVG(price) FROM Products);
```
