## 比较运算符

- `<` (小于)
- `<=` (小于或等于)
- `>` (大于)
- `>=` (大于或等于)
- `=` (等于)
- `<>` (不等于)

任意值与 Null 作比较都是返回 Null

```vb
Debug.Print(True <> Null) ' Null
```

## Not

用于对表达式执行**逻辑非运算**。

|表达式或者位|结果|
|--|--|
|True| False|
|False|True|
|Null|Null|
|0|1|
|1|0|

例子：

```vb
Dim A, B, C, D, MyCheck
A = 10: B = 8: C = 6: D = Null    ' Initialize variables.
MyCheck = Not(A > B)    ' Returns False.
MyCheck = Not(B > A)    ' Returns True.
MyCheck = Not(C > D)    ' Returns Null.
MyCheck = Not A    ' Returns -11 (bitwise comparison). 10: 00001010，-11：11110101(补码)
```
