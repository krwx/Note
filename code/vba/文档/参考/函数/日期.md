## DateSerial 函数

返回指定年月日的 变量（日期）。返回 `Date` 类型的数据

语法：

```vb
DateSerial (year, month, day)
```

例子：

```vb
Dim MyDate
' MyDate contains the date for February 12, 1969.
MyDate = DateSerial(1969, 2, 12)    ' Return a date.
```

### 更改年份

再使用 `DateSerial` 函数（最常用）构建日期

```vb
Dim myDate As Date
myDate = #12/25/2022#

' 将年份改为2023
myDate = DateSerial(2023, Month(myDate), Day(myDate))
' 结果：2023-12-25
```

## Year 函数

返回表示年的整数。

```vb
Dim MyDate, MyYear
MyDate = #February 12, 1969#    ' Assign a date.
MyYear = Year(MyDate)    ' MyYear contains 1969.
```

```vb
Debug.Print Year(Date) ' print 2024
```

## DateAdd 函数

返回一个 Variant (Date) 值，其中包含已添加了指定时间间隔的日期。

语法：`DateAdd(interval, number, date)`

- interval
  - 必需。
  - 作为要添加的时间间隔的字符串表达式。
- number
  - 必需。
  - 作为要添加的时间间隔数的数值表达式。 它可以为正（获取将来日期）或为负（获取过去的日期）。
- date
  - 必需。
  - Variant (Date) 或文本，表示添加间隔的日期。

interval参数具有以下设置：

|Setting|说明|
|--|--|
|yyyy|年|
|q|季度|
|m|月|
|y|每年的某一日|
|d|天|
|w|工作日|
|ww|周|
|h|小时|
|n|分钟|
|s|秒|

向 1 月 10 日添加一个月

```vb
DateAdd("m", 1, "10-Jan-95")
```
