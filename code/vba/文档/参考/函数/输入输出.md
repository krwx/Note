# 输入输出

## Spc()

与 Print # 语句或 Print 方法一起使用来定位输出。

语法：`Spc (n)`

所需的 n参数 是在列表中显示或打印下一个 表达式 之前**要插入的空格数**。

- 如果 n 小于输出行宽度，下一个打印位置将紧跟打印的空格数。 如果 n 大于输出行宽， 则 Spc 使用公式计算下一个打印位置： `current print position + (n Mod 宽度)`。
- 例如，如果当前打印位置为 24，输出行宽度为 80，并且您指定 `Spc (90)` ，则下一个打印将从当前打印位置 (90/80) 的剩余位置 34（24 + 90/80）开始。
- 如果当前打印位置与输出行宽度的差小于 n (或 nMod宽度) ， 则 Spc 函数将跳到下一行的开头，并生成等于 n - (宽度 - 当前打印位置) 的空格。

例子：

```vb
' The Spc function can be used with the Print # statement.
Open "TESTFILE" For Output As #1    ' Open file for output.
Print #1, "10 spaces between here"; Spc(10); "and here."
Close #1    ' Close file.

Debug.Print Spc(30); "Thirty spaces later..."
```

## Tab()

语法：`Tab[ (n) ]`

可选的 n参数 是在列表中显示或打印下一个 表达式 之前移动到的**列号**。 如果省略，则 Tab 会将插入点移到下一个打印区域的开头。 这允许在使用逗号作为小数分隔符的区域设置中使用 Tab 代替逗号。

```vb
' The Tab function can be used with the Print # statement.
Open "TESTFILE" For Output As #1    ' Open file for output.
' The second word prints at column 20.
Print #1, "Hello"; Tab(20); "World."
' If the argument is omitted, cursor is moved to the next print zone.
Print #1, "Hello"; Tab; "World"
Close #1    ' Close file.

Debug.Print Tab(10); "10 columns from start."
```
