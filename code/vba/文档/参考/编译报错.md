# 编译报错

## ByRef argument type mismatch

检查参数类型声明

```vb
' 错误的例子
Sub Test()
    Dim x As String
    x = "123"
    Call ProcessNumber(x)  ' 错误：期望Long，传递了String
End Sub

Sub ProcessNumber(ByRef num As Long)  ' 期望Long类型
    num = num * 2
End Sub

' 正确的做法
Sub Test()
    Dim x As Long
    x = 123
    Call ProcessNumber(x)  ' 正确：传递Long类型
End Sub
```
