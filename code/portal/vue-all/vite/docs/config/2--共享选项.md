## oxc

类型： `OxcOptions | false`

最常见的用例是自定义 JSX：

```js
export default defineConfig({
  oxc: {
    jsx: {
      runtime: 'classic',
      pragma: 'h',
      pragmaFrag: 'Fragment',
    },
  },
})
```

默认情况下，Oxc转换 会被应用在 `ts、jsx、tsx` 文件。你可以通过 `oxc.include` 和 `oxc.exclude` 对要处理的文件类型进行配置。

设置为 `false` 来禁用 Oxc 转换。
