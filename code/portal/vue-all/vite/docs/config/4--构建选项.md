## build.minify

- 类型： `boolean | 'oxc' | 'terser' | 'esbuild'`
- 默认： 客户端构建默认为`'oxc'`，SSR构建默认为 false

设置为 false 可以禁用代码压缩。

默认使用 `Oxc Minifier`，它比 terser 快 30~90 倍，但压缩率仅差 0.5~2%。基准测试

当设置为 `'esbuild'` 或 `'terser'` 时，必须分别安装 esbuild 或 Terser。

```sh
npm add -D esbuild
npm add -D terser
```
