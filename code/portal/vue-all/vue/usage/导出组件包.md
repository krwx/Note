# 导出组件包

组件包结构：

```txt
components/
├── MyButton/
│   ├── index.js
│   └── MyButton.vue
├── index.js
└── package.json
```

`package.json`：

```json
{
  "name": "@my-org/components",
  "version": "1.0.0",
  "main": "index.js",
  "type": "module",
  "license": "ISC"
}
```

`components/index.js`：

这个文件作为组件包的入口，负责导出所有组件

```js
export * from './MyButton'
```

`components/MyButton/index.js`：

这个文件负责导出 `MyButton.vue` 组件

```js
export { default as MyButton } from './MyButton.vue'

// 相当于
import MyButton from './MyButton.vue'
export { MyButton }
```

`MyButton.vue` 组件：

```vue
<!-- MyButton.vue -->
<template>
  <button class="my-btn"><slot /></button>
</template>

<script setup>
// 组件逻辑
</script>
```

这样，我们就可以通过 `import { MyButton } from '@my-org/components'` 来使用 `MyButton` 组件了。

```vue
<script setup>
import { MyButton } from '@my-org/components'
</script>

<template>
  <MyButton>Click me</MyButton>
</template>
```
