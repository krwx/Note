## 使用 vue draggable 实现

[vue draggable 介绍](../../vue%20draggable/vue%20draggable.md)

```vue
<template>
  <div>
    <table border="1" cellpadding="8" style="border-collapse: collapse; width: 100%">
      <thead>
        <tr>
          <th style="width: 50px">拖动</th>
          <th>ID</th>
          <th>姓名</th>
        </tr>
      </thead>
      <!-- 关键：用 draggable 包裹 tbody，并设置 tag="tbody" -->
      <draggable
        tag="tbody"
        item-key="id"
        v-model="items"
        handle=".drag-handle"
        ghost-class="dragging-ghost"
        @end="onDragEnd"
      >
        <template #item="{ element }">
          <tr>
            <td class="drag-handle" style="cursor: grab; text-align: center">☰</td>
            <td>{{ element.id }}</td>
            <td>{{ element.name }}</td>
          </tr>
        </template>
      </draggable>
    </table>

    <pre style="margin-top: 20px">排序结果：{{ items }}</pre>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import draggable from 'vuedraggable'

// 表格数据
const items = ref([
  { id: 1, name: '张三' },
  { id: 2, name: '李四' },
  { id: 3, name: '王五' },
  { id: 4, name: '赵六' }
])

// 拖拽结束时的回调
const onDragEnd = () => {
  console.log('新顺序：', items.value.map(item => item.id))
}
</script>

<style scoped>
/* 拖拽时的半透明样式 */
.dragging-ghost {
  opacity: 0.5;
  background-color: #f0f0f0;
}

/* 可选：表格样式优化 */
table {
  border-collapse: collapse;
  width: 100%;
}
th, td {
  border: 1px solid #ddd;
  padding: 8px;
  text-align: left;
}
</style>
```

## 原生实现

参考 [js 原生实现](../../../js/usage/表格行的拖拽排序.md)

实现思路：

1. 通过 `draggingRowId` 来跟踪当前正在拖拽的行的 ID。
2. 通过 `dropTargetId` 和 `dropPosition` 来跟踪当前的放置目标行的 ID 和放置位置（上方或下方）。
3. 通过绑定 class 来动态设置拖拽行和放置目标行的样式。
   - 当前行为拖拽行时，添加 `dragging` class。
   - 当前行为放置目标行时，根据放置位置添加 `drop-above` 或 `drop-below` class。
4. `onDragstart` 函数设置 `draggingRowId`，表示开始拖拽。
5. `onDragend` 函数清除拖拽状态。
6. `onDragover` 函数根据鼠标位置判断放置位置，并更新 放置目标行（`dropTargetId`） 和 放置位置（`dropPosition`）。
7. `onDrop` 函数根据拖拽行和放置目标行的 ID 来更新 `tableData` 的顺序，并清除拖拽状态。
   - 首先找到拖拽行和放置目标行在 `tableData` 中的索引。
   - 从 `tableData` 中移除拖拽行，此时数组的 index 会发生变化，拖拽行后的行的 index 会减 1。
   - 根据放置位置计算出新的索引，并将拖拽行插入到新的位置。
     - 计算新索引的规则如下：
       - 新的索引初始值为放置目标行的索引。
       - 如果放置位置是下方，则新索引加 1。
       - 如果拖拽行在放置目标行之前，则新索引减 1，因为拖拽行被移除后，放置目标行的索引会减 1。
   - 最后清除拖拽状态，重置 `draggingRowId` 和放置状态。

```vue
<template>
  <div class="wrap">
    <table>
      <thead>
        <tr>
          <th style="width: 80px">序号</th>
          <th>任务</th>
          <th style="width: 140px">负责人</th>
        </tr>
      </thead>
      <tbody id="tableBody">
        <tr 
          v-for="row in tableData" 
          :key="row.id" 
          draggable="true"
          :class="{
            dragging: draggingRowId === row.id,
            'drop-above': dropTargetId === row.id && dropPosition === 'above',
            'drop-below': dropTargetId === row.id && dropPosition === 'below'
          }"
          @dragstart="onDragstart(row.id)"
          @dragend="onDragend"
          @dragover="onDragover($event, row.id)"
          @drop="onDrop($event, row.id)"
        >
          <td>{{ row.id }}</td>
          <td>{{ row.task }}</td>
          <td>{{ row.owner }}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

<script setup>
import { onMounted, ref } from 'vue'

const tableData = ref([
  { id: 1, task: '需求评审', owner: 'Kevin' },
  { id: 2, task: '接口联调', owner: 'Alice' },
  { id: 3, task: '前端开发', owner: 'Bob' },
  { id: 4, task: '测试修复', owner: 'Chris' },
  { id: 5, task: '上线发布', owner: 'Doris' }
])

// 拖拽行的 ID
const draggingRowId = ref(null)
// 放置行的 ID
const dropTargetId = ref(null)
// 放置位置（'above' 或 'below'）
const dropPosition = ref('')

function clearDropState() {
  dropTargetId.value = null
  dropPosition.value = ''
}

const onDragstart = (id) => {
  draggingRowId.value = id
}

const onDragend = () => {
  draggingRowId.value = null
  clearDropState()
}

const onDragover = (event, targetRowId) => {
  if (!draggingRowId.value || draggingRowId.value === targetRowId) {
    return
  }

  event.preventDefault()
  
  const currentRow = event.currentTarget
  const rect = currentRow.getBoundingClientRect()
  const isAbove = event.clientY < rect.top + rect.height / 2

  dropTargetId.value = targetRowId
  dropPosition.value = isAbove ? 'above' : 'below'
}

const onDrop = (event, targetRowId) => {
  event.preventDefault()
  
  if (!draggingRowId.value || draggingRowId.value === targetRowId) {
    return
  }

  const draggingIndex = tableData.value.findIndex(row => row.id === draggingRowId.value)
  const targetIndex = tableData.value.findIndex(row => row.id === targetRowId)

  if (draggingIndex === -1 || targetIndex === -1) {
    clearDropState()
    return
  }

  const draggedItem = tableData.value[draggingIndex]
  tableData.value.splice(draggingIndex, 1)

  let newIndex = targetIndex
  if (dropPosition.value === 'below') {
    newIndex += 1
  }
  if (draggingIndex < targetIndex) {
    newIndex -= 1
  }

  tableData.value.splice(newIndex, 0, draggedItem)
  
  draggingRowId.value = null
  clearDropState()
}

</script>

</style>
```
