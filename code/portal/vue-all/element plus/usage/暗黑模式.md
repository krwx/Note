# 暗黑模式

## 启用

如果您只需要暗色模式，只需在 html 上添加一个名为 `dark` 的类

```html
<html class="dark">
  <head></head>
  <body></body>
</html>
```

如果你想动态切换，创建一个开关来控制 暗黑模式。

```vue
<template>
  <!-- 文字 -->
  <el-switch v-model="isDark" active-text="暗黑模式" inactive-text="浅色模式" @change="toggleDark"/>
  <!-- 图标 -->
  <el-switch v-model="isDark" inline-prompt :active-icon="Moon" :inactive-icon="Sunny" @change="toggleDark()" />
</template>
<script setup>
import { ref, watch } from 'vue'
import { useDark, useToggle } from '@vueuse/core';
import { Sunny, Moon } from '@element-plus/icons-vue';

const isDark = useDark()
const toggleDark = useToggle(isDark)
</script>
```

然后在 `main.ts` 引入暗黑模式的 css 变量文件

```js
import 'element-plus/theme-chalk/dark/css-vars.css'
```

## 自定义 css 变量

直接覆盖对应的 css 变量即可

像这样，新建一个 `styles/dark/css-vars.css` 文件:

```css
html.dark {
  /* 自定义深色背景颜色 */
  --el-bg-color: #626aef;
}
```

在 `main.ts` 文件里面，在 Element Plus 的样式之后导入它

```ts
import 'element-plus/theme-chalk/dark/css-vars.css'
import './styles/dark/css-vars.css'
```
