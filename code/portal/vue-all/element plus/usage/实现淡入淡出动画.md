# 实现淡入淡出动画

使用 Element Plus 实现文本的淡出动画，可以通过 Vue 的 `<transition>` 组件配合 Element Plus 的内置过渡效果（`el-fade-in-linear` 和 `el-fade-in`）完成。下面是一个具体的实现方案。

[官网参考](https://cn.element-plus.org/zh-CN/guide/transitions#fade-%E6%B7%A1%E5%85%A5%E6%B7%A1%E5%87%BA)

## 实现思路

1. **控制文本显示**：使用 Vue 的响应式数据（如 `ref`）控制文本的显示与隐藏。
2. **使用过渡组件**：用 `<transition>` 组件包裹文本，并应用 Element Plus 的淡出过渡效果。
3. **定时隐藏**：点击按钮显示文本，并通过 `setTimeout` 在 5 秒后隐藏文本，触发淡出动画。

### 代码示例

```vue
<template>
  <div>
    <el-button @click="showText">点击显示文本</el-button>
    <transition name="el-fade-in-linear">
      <p v-if="visible" class="demo-text">这段文本将在5秒后淡出</p>
    </transition>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const visible = ref(false);

const showText = () => {
  visible.value = true; // 显示文本
  setTimeout(() => {
    visible.value = false; // 5秒后隐藏文本，触发淡出动画
  }, 5000);
};
</script>

<style scoped>
.demo-text {
  margin-top: 20px;
  font-size: 16px;
  color: #409eff;
}
</style>
```

### 关键点说明

- **过渡效果**：`<transition>` 组件的 `name` 属性设置为 `el-fade-in-linear`，这是 Element Plus 提供的线性淡入淡出效果。也可以使用 `el-fade-in` 获得非线性的淡入淡出效果。
- **显示控制**：通过 `visible` 变量控制文本的显示（`v-if="visible"`）。点击按钮时设为 `true`，5 秒后自动设为 `false`，过渡组件会自动应用淡出动画。
- **动画触发**：文本隐藏时（`visible` 从 `true` 变为 `false`），`<transition>` 会自动应用淡出动画，无需额外配置。

### 其他注意事项

- 如果希望自定义动画时长或缓动函数，可以通过 CSS 覆盖默认样式：

  ```css
  .el-fade-in-linear-enter-active,
  .el-fade-in-linear-leave-active {
    transition: opacity 0.5s ease-in-out;
  }
  ```
