# 工具类型

  - [`Record<Keys, Type>`](#recordkeys-type)

TypeScript 提供了几种工具类型来促进常见的类型转换。这些工具在全局作用域内可用。

## `Required<Type>`

构造一个类型，包含 `Type` 的所有属性，所有属性变为**必需**。

```ts
interface Props {
  a?: number;
  b?: string;
}
 
const obj: Props = { a: 5 };
 
const obj2: Required<Props> = { a: 5 };
// 提示错误：Property 'b' is missing in type '{ a: number; }' but required in type 'Required<Props>'.
```

## `Record<Keys, Type>`

构造一个对象类型，具有一组属性 `Keys`，属性值类型为 `Type`。

```ts
type Page = 'home' | 'about' | 'contact';
type PageInfo = Record<Page, { title: string }>;
// 等价于
type PageInfo = {
  home: { title: string };
  about: { title: string };
  contact: { title: string };
};

// 使用
const pageInfo: PageInfo = {
  home: { title: 'Home Page' },
  about: { title: 'About Us' },
  contact: { title: 'Contact Us' },
};
```

如果对象缺少了 `Record` 定义的属性，或者属性值类型不匹配，都会提示错误：

```ts
const invalidPageInfo: PageInfo = {
  home: { title: 'Home Page' },
  about: { title: 'About Us' },
  // 错误：Property 'contact' is missing in type '{ home: { title: string; }; about: { title: string; }; }' but required in type 'PageInfo'.
};
```

## `ReturnType<Type>`

> `<Type>` 中的 `Type` 是一个函数

构造一个类型，它提取了函数返回值的类型。

```ts
type T0 = ReturnType<() => string>;
// () => string 这个箭头函数返回 string
// type T0 = string
     
type T1 = ReturnType<(s: string) => void>; // type T1 = void
     
type T2 = ReturnType<<T>() => T>; // type T2 = unknown

type T3 = ReturnType<<T extends U, U extends number[]>() => T>; // type T3 = number[]
     
declare function f1(): { a: number; b: string };
type T4 = ReturnType<typeof f1>;
/* 
相当于
type T4 = {
    a: number;
    b: string;
}
*/
```

使用例子：

```ts
// 例子1
function greet() {
  return "hello";
}
type GreetReturn = ReturnType<typeof greet>;  // string


// 例子2
const add = (a: number, b: number) => a + b;
type AddReturn = ReturnType<typeof add>;      // number


// 例子3
async function fetchData() {
  return { id: 1 };
}
type FetchReturn = ReturnType<typeof fetchData>; // Promise<{ id: number }>
```
