## 创建一个正则表达式

```javascript
// 方式一：使用字面量语法
const regex1 = /ab+c/;
```

```javascript
// 方式二：使用 RegExp 构造函数
const regex2 = new RegExp('ab+c');
```

## 使用正则表达式

正则表达式可以被用于 `RegExp` 的 `exec` 和 `test` 方法以及 `String` 的 `match`、`replace`、`search` 和 `split` 方法。

### RegExp

|方法|描述|
|--|--|
|`exec`|一个在字符串中执行**查找匹配**的 RegExp 方法，它返回一个数组（未匹配到则返回 null），数组中包含匹配的结果以及捕获的子组。|
|`test`|一个在字符串中**测试是否匹配**的 RegExp 方法，它返回 true 或 false。|

#### exec()

**1、返回值**

- **匹配成功**：返回一个数组，并额外附加 `index`、`input` 和 `groups` 属性。
- **匹配失败**：返回 `null`。

**2、返回数组的内容**

| 索引/属性 | 说明 |
| ----------- | ------ |
| `[0]` | 本次匹配到的完整子串 |
| `[1]` … `[n]` | 第 1 到第 n 个捕获组（如果有） |
| `index` | 匹配子串在原字符串中的起始索引（从 0 开始） |
| `input` | 原始目标字符串（即 `exec` 的参数） |
| `groups` | 一个对象，存储命名捕获组（如果使用了 `(?<name>…)`） |

```js
const re = /(\d{4})-(\d{2})-(\d{2})/;
const str = '日期：2025-12-31';
const result = re.exec(str);

console.log(result);
// 输出：
// [
//   '2025-12-31',   // 完整匹配
//   '2025',         // 第一个捕获组
//   '12',           // 第二个捕获组
//   '31',           // 第三个捕获组
//   index: 3,       // 匹配起始位置
//   input: '日期：2025-12-31',
//   groups: undefined
// ]
```

**3、与 `/g`（全局标志）的配合**

- **没有 `/g`**：`exec()` 每次调用都只返回第一个匹配，`lastIndex` 属性被忽略。
- **有 `/g`**：`exec()` 会在每次调用后更新正则对象的 `lastIndex` 属性（即下一次搜索的起始位置）。可以循环调用直到返回 `null`，从而遍历所有匹配。

```js
const str = 'apple, banana, apple';
const re = /apple/g;

let match;
while ((match = re.exec(str)) !== null) {
  console.log(`找到 "${match[0]}"，位置 ${match.index}`);
}
// 输出：
// 找到 "apple"，位置 0
// 找到 "apple"，位置 14
```

注意：要正则表达式有 `/g` 才能用循环的方式找到所有匹配项，否则 `exec()` 每次调用都会返回同一个匹配结果，导致死循环。

```js
const str = 'apple, banana, apple';
const re = /apple/;

let match;
while ((match = re.exec(str)) !== null) { // 这里会死循环
  console.log(`找到 "${match[0]}"，位置 ${match.index}`);
}
```

**3.1、手动重置 `lastIndex`**

使用全局正则多次匹配时，务必注意 `lastIndex` 的残留问题。使用了 `exec()` 后就相当于匹配了一次，`lastIndex` 会自动更新。例如：

```js
const re = /test/g;
re.exec('test test');   // 匹配成功，lastIndex = 4
re.exec('test test');   // 第二次调用从索引 4 开始，仍然匹配成功（第二个 test），lastIndex = 9
re.exec('test test');   // 第三次调用从 9 开始，返回 null，lastIndex 自动重置为 0
```

如果需要重新开始搜索，可以显式设置 `lastIndex = 0`：

```js
re.lastIndex = 0;
```

> 同一个正则对象在多次调用 `exec()` 时 `lastIndex` 会累积。如果要在不同字符串上使用同一个正则对象，记得重置 `lastIndex`

**4、命名捕获组示例**

```js
const re = /(?<year>\d{4})-(?<month>\d{2})/;
const result = re.exec('2025-03');
console.log(result.groups);   // { year: '2025', month: '03' }
```

### String

|方法|描述|
|--|--|
|`match`|一个在字符串中执行查找匹配的 String 方法，它返回一个包含匹配到的字符串的数组，在未匹配到时会返回 null。没有 `/g` 标志时，只返回第一个匹配和捕获组信息；有 `/g` 标志时，返回所有匹配，丢失捕获组。|
|`matchAll`|一个在字符串中执行查找所有匹配的 String 方法，它返回一个迭代器（iterator）。|
|`search`|一个在字符串中测试匹配的 String 方法，它**返回匹配到的位置索引**，或者在失败时返回 -1。|
|`replace`|一个在字符串中执行查找匹配的 String 方法，并且使用替换字符串替换掉匹配到的子字符串。|
|`split`|一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组中的 String 方法。|

#### match()

1. 没有全局标志 `g`
   - 返回一个数组，数组的第一个元素是第一个完整匹配的字符串。
   - 如果正则表达式包含捕获组，数组中还会包含捕获组匹配的子串（从索引 1 开始）。
   - 该数组额外带有 `index` 属性（匹配的起始位置）、`input` 属性（原始字符串）和 `groups` 属性（命名捕获组，若有）。
2. 有全局标志 `g`
   - 返回一个数组，包含所有匹配的子串（不包括捕获组）。
   - 没有 `index`、`input` 等额外属性。

```js
let str1 = "abc  abbc acb";
let reg1 = /ab*/;
console.log(str1.match(reg1)); // 输出: [ 'ab', index: 0, input: 'abc  abbc acb', groups: undefined ]

let str1 = "abc  abbc acb";
let reg1 = /ab*/g;
console.log(str1.match(reg1)); // 输出: ["ab", "abb", "ab"]
```

#### matchAll()

正则表达式一定要带 `/g` 标志，否则会抛出 `TypeError` 错误。

```js
let str1 = "abc  abbc acb";
let reg1 = /ab*/g;
let matches = str1.matchAll(reg1);
for (const match of matches) {
  console.log(match);
}
// 输出:
// [ 'ab', index: 0, input: 'abc  abbc acb', groups: undefined ]
// [ 'abb', index: 5, input: 'abc  abbc acb', groups: undefined ]
// [ 'ab', index: 10, input: 'abc  abbc acb', groups: undefined ]
```

#### search()

```js
let str1 = "bbc abbc acb";
let reg1 = /ab*/;
console.log(str1.search(reg1)); // 输出: 4
```

#### replace()

```js
let str1 = "abc abbc acb";
let reg1 = /ab*/g;
console.log(str1.replace(reg1, "X")); // 输出: "Xc Xc Xcb"
```

### 使用括号的子字符串匹配

一个正则表达式模式使用括号，将导致相应的子匹配被记住。

例如，`/a(b)c /`可以匹配字符串`“abc”`，并且记得`“b”`。回调这些括号中匹配的子串，使用数组元素 `[1],……[n]`。

下面的脚本使用 `replace()` 方法来转换字符串中的单词。在匹配到的替换文本中，脚本使用替代的 `$1`,`$2` 表示第一个和第二个括号的子字符串匹配。

```js
var re = /(\w+)\s(\w+)/;
var str = "John Smith";
var newstr = str.replace(re, "$2, $1");
console.log(newstr); // Output: "Smith, John"
```

### 通过标志进行高级搜索

|标志|描述|
|--|--|
|g|全局搜索。|
|i|不区分大小写搜索。|
|m|多行搜索。|
|s|允许 `.` 匹配换行符。|
|u|使用 unicode 码的模式进行匹配。|
|y|执行“粘性 (`sticky`)”搜索，匹配从目标字符串的当前位置开始。|

**没有加 g 作为修饰符的匹配，找到第一个匹配的地方就会停止；加了 g 作为修饰符的匹配，会查找所有匹配，直到字符串结束。**
