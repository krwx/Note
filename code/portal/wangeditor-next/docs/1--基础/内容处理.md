## 获取内容

声明一个 `ref`，然后通过 `v-model` 绑定到 `Editor` 上，就可以获取到编辑器的 HTML 内容了。

```vue
<template>
  <div style="border: 1px solid #ccc">
    <Toolbar :editor="editorRef" :defaultConfig="toolbarConfig"/>
    <Editor v-model="htmlBody" :defaultConfig="editorConfig"/>
  </div>
</template>

<script>
import { ref } from 'vue'
import { Editor, Toolbar } from '@wangeditor-next/editor-for-vue'
const editorRef = ref(null)
const toolbarConfig = {
  // ...
}
const editorConfig = {
  // ...
}

const htmlBody = ref('')
const someFunction = () => {
  console.log(htmlBody.value) // 获取 HTML 内容
}
</script>
```

## 设置内容

更改上面获取内容定义的 `ref` 的值，就可以设置编辑器的内容了。

`HTML` 的格式需要是 `wangEditor` 支持的格式（可以通过查看 `ref` 的值来确认）。

例如，`wangEditor` 可以识别 `<strong>hello</strong>` 为加粗，但无法识别 `<span style="font-weight: bold;">hello</span>` 等其他加粗方式。

```vue
<template>
  <div style="border: 1px solid #ccc">
    <Toolbar :editor="editorRef" :defaultConfig="toolbarConfig"/>
    <Editor v-model="htmlBody" :defaultConfig="editorConfig"/>
  </div>
</template>

<script>
import { ref } from 'vue'
import { Editor, Toolbar } from '@wangeditor-next/editor-for-vue'
const editorRef = ref(null)
const toolbarConfig = {
  // ...
}
const editorConfig = {
  // ...
}

const htmlBody = ref('')
const someFunction = () => {
  htmlBody.value = '<p>这是新的内容</p>' // 设置 HTML 内容
}
</script>
```

### table

wangeditor 的 table 的 `table-layout` 属性是 `auto`，所以表格的宽度是根据内容自动调整的。

但是 wangeditor-next 的 table 的 `table-layout` 属性是 `fixed`，所以表格的宽度是固定的（90px），不会根据内容自动调整。

所以当设置内容时需要我们自己设置 `th` 的宽度，例如：

```ts
htmlBody.value = '<table><tr><th style="width: 100px;">名字</th><th style="width: 80px;">年龄</th></tr><tr><td>张三</td><td>18</td></tr></table>'
```
