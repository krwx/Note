## 水平居中

（1）我们可以利用 `margin:0 auto` 来实现元素的水平居中。

```css
div {
  width: 200px;
  margin: 0 auto;
}
```

（2）水平居中，利用 `text-align:center` 实现

```css
.container {
  background: rgba(0, 0, 0, 0.5);
  text-align: center;
  font-size: 0;
}

.box {
  display: inline-block;
  width: 500px;
  height: 400px;
  background-color: pink;
}
```

## 水平垂直居中

### 1、使用 `flex` 布局

通过 `align-items:center` 和 `justify-content:center` 设置容器的垂直和水平方向上为居中对齐，然后它的子元素也可以实现垂直和水平的居中。

```html
<div class="parent">
  <div class="child">居中的内容</div>
</div>
```

```css
.parent {
  display: flex;
  justify-content: center; /* 水平居中 */
  align-items: center;     /* 垂直居中 */
  height: 100vh;           /* 父容器必须有高度，例如占满视口 */
}
```

### 2、使用 `grid` 布局

通过 `place-items:center` 设置容器的垂直和水平方向上为居中对齐，然后它的子元素也可以实现垂直和水平的居中。

```css
.parent {
  display: grid;
  place-items: center;     /* 同时设置水平垂直居中 */
  height: 100vh;
}
```

说明：`place-items` 是 `align-items` 和 `justify-items` 的简写，同时控制所有子项居中。

### 3、绝对定位 + transform（适用于未知宽高）

子元素设置绝对定位，先将元素的左上角通过 `top:50%` 和 `left:50%` 定位到页面的中心，然后再通过 `transform` 平移自身宽高的一半来调整元素的中心点到页面的中心。

```css
.parent {
  position: relative;
  height: 100vh;
}

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%); /* 向左向上移动自身宽高的一半 */
}
```

### 4、绝对定位 + margin（需固定宽高）

利用绝对定位，先将元素的左上角通过 `top:50%` 和 `left:50%` 定位到页面的中心，然后再通过 `margin` 负值来调整元素的中心点到页面的中心。

```css
div {
	position: absolute;/*绝对定位*/
	width: 500px;
	height: 300px;
	top: 50%;
	left: 50%;
	margin: -150px 0 0 -250px;/*外边距为自身宽高的一半*/
	background-color: pink;/*方便看效果*/
}
```

### 5、绝对定位 + `margin: auto`（需固定宽高）

如果子元素有固定宽高，可以通过设置四个方向为 0，配合 `margin: auto` 实现。

```css
.parent {
  position: relative;
  height: 100vh;
}

.child {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  width: 200px;        /* 必须指定宽度 */
  height: 150px;       /* 必须指定高度 */
  margin: auto;        /* 自动分配外边距 */
}
```

原理：在绝对定位下，四个方向坐标设置为 0，`margin: auto` 会使浏览器平均分配剩余空间，从而实现居中。

### 6、使用 table-cell（传统方式）

将父容器模拟为表格单元格，利用 `vertical-align` 和 `text-align` 居中。子元素需为行内块或行内元素。

```css
.parent {
  display: table-cell;
  text-align: center;      /* 水平居中 */
  vertical-align: middle;  /* 垂直居中 */
  width: 100vw;
  height: 100vh;
}

.child {
  display: inline-block;   /* 让子元素可以水平居中，且不影响垂直居中 */
}
```

缺点：需要父容器设置 `display: table-cell`，且宽度高度必须明确，不够灵活，现在较少使用。
