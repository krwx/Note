## 实现两栏

```html
	<body>
		<div class="outer">
			<div class="left">123</div>
			<div class="right">right</div>
		</div>
	</body>
```

（1）利用**浮动**，将左边元素宽度设置为200px，并且设置向左浮动。将右边元素的margin-left设置为200px，宽度设置为auto（默认为auto，撑满整个父元素）

```css
.outer {
  height: 100px;
}

.left {
  float: left;

  height: 100px;
  width: 200px;

  background: tomato;
}

.right {
  margin-left: 200px;

  width: auto;
  height: 100px;

  background: gold;
}
```

（2）第二种是利用 **flex 布局**，将左边元素的放大和缩小比例设置为 0，基础大小设置为 200px 。将右边的元素的放大比例设置为 1 ，缩小比例设置为 1 ，基础大小设置为auto。

```css
.outer {
  display: flex;

  height: 100px;
}

.left {
  flex-shrink: 0;
  flex-grow: 0;
  flex-basis: 200px;

  background: tomato;
}

.right {
  flex: auto;
  /*11auto*/

  background: gold;
}
```

（3）第三种是利用 **绝对定位** 布局的方式，  
将父级元素设置相对定位。左边元素设置为 absolute 定位，并且宽度设置为 200px。将右边元素的 margin-left 的值设置为200px。

```css
.outer {
  position: relative;

  height: 100px;
}

.left {
  position: absolute;

  width: 200px;
  height: 100px;

  background: tomato;
}

.right {
  margin-left: 200px;
  height: 100px;

  background: gold;
}
```

（4）第四种还是利用**绝对定位**的方式，  
将父级元素设置为相对定位。左边元素宽度设置为 200px，**右边元素设置为绝对定位**，左边定位为 200px，其余方向定位为0。

```css
.outer {
  position: relative;

  height: 100px;
}

.left {
  width: 200px;
  height: 100px;

  background: tomato;
}

.right {
  position: absolute;

  top: 0;
  right: 0;
  bottom: 0;
  left: 200px;

  background: gold;
}
```

## 实现三栏

```html
	<body>
		<div class="outer">
			<div class="left">123</div>
			<div class="center">
				center
			</div>
			<div class="right">
				<div>
					<button>test</button>
				</div>
			</div>
		</div>
	</body>
```

(5) 圣杯布局。利用**浮动和负边距**来实现。

- 父级元素设置 `padding-left` 为左边元素的宽度，设置 `padding-right` 为右边元素的宽度
- 左边和中间均设置向左浮动，右边设置向右浮动。**中间一列放在最前面**，宽度设置为父级元素的宽度，因此后面两列都被挤到了下一行。
- 通过设置 margin 负值将其移动到上一行，
- 再利用相对定位，定位到两边

```html
		<div class="outer">
			<div class="center">
				center
			</div>
			<div class="left">123</div>
			<div class="right">
				<div>
					<button>test</button>
				</div>
			</div>
		</div>
```

```css
.outer {
  height: 100px;
  padding-left: 100px;
  padding-right: 200px;
}

.left {
  position: relative;
  left: -100px;  /* 因为父级元素设置了 padding，所以需要设置 left 移动到边框处 */

  float: left;
  margin-left: -100%;  /* 移动到上一行 */

  width: 100px;
  height: 100px;
  background: tomato;
}

.right {
  position: relative;
  left: 200px;

  float: right;
  margin-left: -200px;

  width: 200px;
  height: 100px;
  background: gold;
}

.center {
  float: left;

  width: 100%;
  height: 100px;
  background: lightgreen;
}
```

(6) 双飞翼布局

- 双飞翼布局相对于圣杯布局来说，左右位置的保留是通过中间列的 margin 值来实现的。  
- 即 wrapper 占一行，left 和 right 都是下一行的，但是通过 float 、 position: relative 和 margin 等设置移动到上一行。
- center 通过设置 margin 来产生左右两边的空间

```html
<div class="outer">
    <div class="wrapper">
      <div class="center">圣杯-middle</div>
    </div>
    <div class="left">圣杯-left</div>
    <div class="right">圣杯-right</div>
</div>
```

```css
.outer {
  height: 100px;
}

.left {
  float: left;
  margin-left: -100%;

  width: 100px;
  height: 100px;
  background: tomato;
}

.right {
  float: left;
  margin-left: -200px;

  width: 200px;
  height: 100px;
  background: gold;
}

.wrapper {
  float: left;

  width: 100%;
  height: 100px;
  background: lightgreen;
}

.center {
  margin-left: 100px;
  margin-right: 200px;
  height: 100px;
}
```

### 1. **浮动布局 (Float)**

利用 `float` 让左右两栏浮动，中间栏通过 `margin` 避开左右栏。  
**注意**：需要修改 HTML 结构，将中间栏放在最后，以保证左浮动的正常排列。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
  <div class="middle">中间自适应</div>
</div>
```

```css
.left {
  float: left;
  width: 200px;
  background: lightblue;
}
.right {
  float: right;
  width: 200px;
  background: lightgreen;
}
.middle {
  margin-left: 200px;   /* 等于左栏宽度 */
  margin-right: 200px;  /* 等于右栏宽度 */
  background: lightgray;
}
```

**优点**：兼容性好。  
**缺点**：需要清除浮动，且中间栏必须写在最后，对 SEO 不友好。

### 2. **绝对定位 (Absolute Positioning)**

#### 左右两栏绝对定位

父容器相对定位，左右两栏绝对定位，中间用 `margin` 撑开。

1. 左栏设置 `left: 0`，右栏设置 `right: 0`，都设置 `top: 0` 和 `bottom: 0` 来占满父容器高度。
2. 中间栏设置 `margin-left` 为左栏的宽度，`margin-right` 为右栏的宽度

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  position: relative;
}
.left, .right {
  position: absolute;
  top: 0;
  width: 200px;
}
.left {
  position: absolute;
  width: 100px;
  left: 0;
  background: lightblue;
}
.right {
  position: absolute;
  width: 200px;
  right: 0;
  background: lightgreen;
}
.middle {
  margin-left: 100px;
  margin-right: 200px;
  background: lightgray;
}
```

**优点**：结构简单，任意顺序。  
**缺点**：父容器高度依赖中间栏，若左右栏内容更多可能溢出。

#### 中间栏和右栏绝对定位

中间栏和右栏设置 `position: absolute`

1. 中间栏设置 left 为左栏的宽度，right 为右栏的宽度，top 和 bottom 设置为 0
2. 右栏设置 right 为 0，top 和 bottom 设置为 0

```css
		.outer {
			height: 200px;
			position: relative;
		}
		.left {
			height: 100%;
			background-color: antiquewhite;
			width: 150px
		}
		.right {
			background-color: red;
			height: 100%;
			width: 80px;
			
			position: absolute;
			top: 0;
			right: 0;
		}
		.center {
			background-color: dodgerblue;
			
			position: absolute;
			left: 150px;
			top: 0;
			bottom: 0;
			right: 80px;
		}
```

### 3. **Flexbox 布局 (推荐)**

中间一栏设置 `flex: auto` 或 `flex: 1`。左右两栏设置 `flex: 0 0 固定宽度` 或只设置 width 。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  display: flex;
}
.left {
  width: 100px;
  /* or */
  flex: 0 0 100px;
  background: lightblue;
}
.right {
  width: 200px;
  background: lightgreen;
}
.middle {
  flex: 1;           /* 占据剩余空间 */
  background: lightgray;
}
```

**优点**：简单、强大，自动等高。  
**缺点**：浏览器兼容性良好，可放心使用（IE10+）。

### 4. **Grid 布局 (最新潮)**

CSS Grid 是二维布局系统，实现三栏布局尤为直观。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px; /* 左右固定，中间自适应 */
  gap: 10px; /* 可选间距 */
}
.left {
  background: lightblue;
}
.middle {
  background: lightgray;
}
.right {
  background: lightgreen;
}
```

**优点**：代码极简，语义清晰。  
**缺点**：较新，但现代浏览器支持良好（IE 不支持）。

### 5. **圣杯布局 (Holy Grail) 与双飞翼布局**

这两种是经典的浮动改进方案，主要解决中间栏优先渲染（先出现在 HTML 中）且三列等高的问题。

#### 圣杯布局

利用 `padding` 腾出空间，左右栏通过相对定位移动位置。

思路：

1. **浮动建立基础列**
   - 三栏均设置 `float: left`，使它们从左到右排列。
   - 中间栏宽度设为 `width: 100%`，独占父容器一整行。
   - 左右栏宽度固定（例如 `200px`）。
   - 此时，由于中间栏占满整行，左右栏会被“挤”到下一行。
2. **负外边距将左右栏拉回上一行**
   - 左栏设置 `margin-left: -100%`。`-100%` 相对于父容器宽度，相当于将左栏向左移动一整行的距离，使其正好浮动到中间栏的最左边（与中间栏重叠）。
   - 右栏设置 `margin-left: -200px`（自身宽度的负值）。这使其向左移动自身宽度，正好浮动到中间栏的最右边（同样与中间栏重叠）。
   - 此时三栏已经全部堆叠在中间栏占据的那一行上，左右栏覆盖在中间栏的两端。
3. **父容器内边距为左右栏腾出空间**
   - 父容器设置 `padding-left: 200px; padding-right: 200px;`，左右内边距分别等于左右栏宽度。
   - 这样中间栏的内容区域被向内挤压，左右两侧空出了位置，但左右栏目前仍覆盖在中间栏之上。
4. **相对定位将左右栏移入预留区域**
   - 父容器设置 `position: relative` 作为定位参照。
   - 左栏设置 `position: relative; left: -200px;`，将其向左拉出父容器的内边距区域，最终定位在左侧预留位置。
   - 右栏设置 `position: relative; right: -200px;`（或 `left: 200px;`），将其向右移动，定位在右侧预留位置。
   - 此时左右栏从覆盖状态分别移到了两侧的内边距区域，中间栏自适应宽度，布局完成。

```html
<div class="container">
  <div class="middle">中间自适应</div>
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  padding-left: 200px;
  padding-right: 200px;
}
.middle {
  float: left;
  width: 100%;
  background: lightgray;
}
.left {
  float: left;
  width: 200px;
  margin-left: -100%;
  position: relative;
  left: -200px;
  background: lightblue;
}
.right {
  float: left;
  width: 200px;
  margin-right: -200px;
  background: lightgreen;
}
```

#### 双飞翼布局

在圣杯基础上，中间栏内部再加一层 `inner`，通过 `margin` 避开左右栏，省去了定位。

```html
<div class="container">
  <div class="middle">
    <div class="inner">中间自适应</div>
  </div>
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.middle {
  float: left;
  width: 100%;
}
.inner {
  margin: 0 200px; /* 左右栏宽度 */
  background: lightgray;
}
.left {
  float: left;
  width: 200px;
  margin-left: -100%;
  background: lightblue;
}
.right {
  float: left;
  width: 200px;
  margin-left: -200px;
  background: lightgreen;
}
```
