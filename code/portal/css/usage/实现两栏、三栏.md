## 实现两栏

```html
	<body>
		<div class="outer">
			<div class="left">123</div>
			<div class="right">right</div>
		</div>
	</body>
```

### 1. 浮动 + margin

利用**浮动**，将左边元素宽度设置为 200px，并且设置向左浮动。将右边元素的 `margin-left` 设置为 200px，宽度设置为 auto（默认为auto，撑满整个父元素）

```css
.outer {
  height: 100px;
}

.left {
  float: left;

  height: 100px;
  width: 200px;

  background: tomato;
}

.right {
  margin-left: 200px;

  width: auto;
  height: 100px;

  background: gold;
}
```

### 2. Flex 布局

第二种是利用 **flex 布局**，将左边元素的放大和缩小比例设置为 0，基础大小设置为 200px 。将右边的元素的放大比例设置为 1 ，缩小比例设置为 1 ，基础大小设置为auto。

```css
.outer {
  display: flex;

  height: 100px;
}

.left {
  flex-shrink: 0;
  flex-grow: 0;
  flex-basis: 200px;

  background: tomato;
}

.right {
  flex: auto;
  /*11auto*/

  background: gold;
}
```

### 3. 绝对定位

#### 左边元素设置绝对定位

将父级元素设置相对定位。左边元素设置为 `absolute` 定位，并且宽度设置为 200px。将右边元素的 `margin-left` 的值设置为 200px。

```css
.outer {
  position: relative;

  height: 100px;
}

.left {
  position: absolute;

  width: 200px;
  height: 100px;

  background: tomato;
}

.right {
  margin-left: 200px;
  height: 100px;

  background: gold;
}
```

#### 右边元素设置绝对定位

将父级元素设置为相对定位。左边元素宽度设置为 200px，**右边元素设置为绝对定位**，左边定位为 200px，其余方向定位为0。

```css
.outer {
  position: relative;

  height: 100px;
}

.left {
  width: 200px;
  height: 100px;

  background: tomato;
}

.right {
  position: absolute;

  top: 0;
  right: 0;
  bottom: 0;
  left: 200px;

  background: gold;
}
```

### 4. grid 布局

CSS Grid 是二维布局系统，实现两栏布局同样简洁。

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr;  /* 第一列200px，第二列占据剩余空间 */
}

.left {
  background: #f0f0f0;
  padding: 10px;
}

.right {
  background: #d9d9d9;
  padding: 10px;
}
```

说明：

- `1fr` 表示剩余空间的一份，效果等同于 `flex: 1`。
- 网格布局可以方便地控制列间距（gap）和行对齐。

## 实现三栏

```html
	<body>
		<div class="outer">
			<div class="left">123</div>
			<div class="center">
				center
			</div>
			<div class="right">
				<div>
					<button>test</button>
				</div>
			</div>
		</div>
	</body>
```

### 1. **浮动布局 (Float)**

利用 `float` 让左右两栏浮动，中间栏通过 `margin` 避开左右栏。  
**注意**：需要修改 HTML 结构，将中间栏放在最后，以保证左浮动的正常排列。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
  <div class="middle">中间自适应</div>
</div>
```

```css
.left {
  float: left;
  width: 200px;
  background: lightblue;
}
.right {
  float: right;
  width: 200px;
  background: lightgreen;
}
.middle {
  margin-left: 200px;   /* 等于左栏宽度 */
  margin-right: 200px;  /* 等于右栏宽度 */
  background: lightgray;
}
```

**优点**：兼容性好。  
**缺点**：需要清除浮动，且中间栏必须写在最后，对 SEO 不友好。

### 2. **绝对定位 (Absolute Positioning)**

#### 左右两栏绝对定位

父容器相对定位，左右两栏绝对定位，中间用 `margin` 撑开。

1. 左栏设置 `left: 0`，右栏设置 `right: 0`，都设置 `top: 0` 和 `bottom: 0` 来占满父容器高度。
2. 中间栏设置 `margin-left` 为左栏的宽度，`margin-right` 为右栏的宽度

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  position: relative;
}
.left, .right {
  position: absolute;
  top: 0;
  width: 200px;
}
.left {
  position: absolute;
  width: 100px;
  left: 0;
  background: lightblue;
}
.right {
  position: absolute;
  width: 200px;
  right: 0;
  background: lightgreen;
}
.middle {
  margin-left: 100px;
  margin-right: 200px;
  background: lightgray;
}
```

**优点**：结构简单，任意顺序。  
**缺点**：父容器高度依赖中间栏，若左右栏内容更多可能溢出。

#### 中间栏和右栏绝对定位

中间栏和右栏设置 `position: absolute`

1. 中间栏设置 left 为左栏的宽度，right 为右栏的宽度，top 和 bottom 设置为 0
2. 右栏设置 right 为 0，top 和 bottom 设置为 0

```css
		.outer {
			height: 200px;
			position: relative;
		}
		.left {
			height: 100%;
			background-color: antiquewhite;
			width: 150px
		}
		.right {
			background-color: red;
			height: 100%;
			width: 80px;
			
			position: absolute;
			top: 0;
			right: 0;
		}
		.center {
			background-color: dodgerblue;
			
			position: absolute;
			left: 150px;
			top: 0;
			bottom: 0;
			right: 80px;
		}
```

### 3. **Flexbox 布局 (推荐)**

中间一栏设置 `flex: auto` 或 `flex: 1`。左右两栏设置 `flex: 0 0 固定宽度` 或只设置 width 。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  display: flex;
}
.left {
  width: 100px;
  /* or */
  flex: 0 0 100px;
  background: lightblue;
}
.right {
  width: 200px;
  background: lightgreen;
}
.middle {
  flex: 1;           /* 占据剩余空间 */
  background: lightgray;
}
```

**优点**：简单、强大，自动等高。  
**缺点**：浏览器兼容性良好，可放心使用（IE10+）。

### 4. **Grid 布局 (最新潮)**

CSS Grid 是二维布局系统，实现三栏布局尤为直观。

```html
<div class="container">
  <div class="left">左栏 200px</div>
  <div class="middle">中间自适应</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px; /* 左右固定，中间自适应 */
  gap: 10px; /* 可选间距 */
}
.left {
  background: lightblue;
}
.middle {
  background: lightgray;
}
.right {
  background: lightgreen;
}
```

**优点**：代码极简，语义清晰。  
**缺点**：较新，但现代浏览器支持良好（IE 不支持）。

### 5. **圣杯布局 (Holy Grail) 与双飞翼布局**

这两种是经典的浮动改进方案，主要解决中间栏优先渲染（先出现在 HTML 中）且三列等高的问题。

#### 圣杯布局（float、padding、margin、relative position）

利用 `padding` 腾出空间，左右栏通过相对定位移动位置。

思路：

1. **浮动建立基础列**
   - 三栏均设置 `float: left`，使它们从左到右排列。
   - 中间栏宽度设为 `width: 100%`，独占父容器一整行。
   - 左右栏宽度固定（例如 `200px`）。
   - 此时，由于中间栏占满整行，左右栏会被“挤”到下一行。
   ![Holy Grail1](../img/Holy%20Grail-1.png)
2. **负外边距将左右栏拉回上一行**
   - 左栏设置 `margin-left: -100%`。`-100%` 相对于父容器宽度，相当于将左栏向左移动一整行的距离，使其正好浮动到中间栏的最左边（与中间栏重叠）。
   - 右栏设置 `margin-left: -200px`（自身宽度的负值）。这使其向左移动自身宽度，正好浮动到中间栏的最右边（同样与中间栏重叠）。
   - 此时三栏已经全部堆叠在中间栏占据的那一行上，左右栏覆盖在中间栏的两端。
   ![Holy Grail2](../img/Holy%20Grail-2.png)
3. **父容器内边距为左右栏腾出空间**
   - 父容器设置 `padding-left: 200px; padding-right: 200px;`，左右内边距分别等于左右栏宽度。
   - 这样中间栏的内容区域被向内挤压，左右两侧空出了位置，但左右栏目前仍覆盖在中间栏之上。
   ![Holy Grail3](../img/Holy%20Grail-3.png)
4. **相对定位将左右栏移入预留区域**
   - 父容器设置 `position: relative` 作为定位参照。
   - 左栏设置 `position: relative; left: -200px;`，将其向左拉出父容器的内边距区域，最终定位在左侧预留位置。
   - 右栏设置 `position: relative; right: -200px;`（或 `left: 200px;`），将其向右移动，定位在右侧预留位置。
   - 此时左右栏从覆盖状态分别移到了两侧的内边距区域，中间栏自适应宽度，布局完成。
   ![Holy Grail4](../img/Holy%20Grail-4.png)

```html
<div class="container">
  <div class="middle">中间自适应</div>
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.container {
  padding-left: 200px;
  padding-right: 200px;
}
.middle {
  float: left;
  width: 100%;
  background: lightgray;
}
.left {
  position: relative;
  left: -200px;

  float: left;
  margin-left: -100%;

  width: 200px;
  background: lightblue;
}
.right {
  position: relative;
  left: 200px;
  /* or */
  right: -200px;

  float: left;
  width: 200px;

  margin-left: -200px;
  background: lightgreen;
}
```

#### 双飞翼布局（float、padding、margin）

在圣杯基础上，中间栏内部再加一层 `inner`，通过 `margin` 避开左右栏，省去了定位。

思路：

1. **HTML 结构调整**
   - 将中间栏拆分为两层：外层 `middle` 和内层 `inner`。
   - `middle` 负责占位和浮动，`inner` 负责内容（背景颜色）和左右栏的间距。
2. **浮动建立基础列**
   - 三栏均设置 `float: left`，使它们从左到右排列。
   - 中间栏宽度设为 `width: 100%`，独占父容器一整行。
   - 左右栏宽度固定（例如 `200px`）。
   - 此时，由于中间栏占满整行，左右栏会被“挤”到下一行。
   ![Holy Grail1](../img/Holy%20Grail-1.png)
3. **负外边距将左右栏拉回上一行**
   - 左栏设置 `margin-left: -100%`。`-100%` 相对于父容器宽度，相当于将左栏向左移动一整行的距离，使其正好浮动到中间栏的最左边（与中间栏重叠）。
   - 右栏设置 `margin-left: -200px`（自身宽度的负值）。这使其向左移动自身宽度，正好浮动到中间栏的最右边（同样与中间栏重叠）。
   - 此时三栏已经全部堆叠在中间栏占据的那一行上，左右栏覆盖在中间栏的两端。
4. **中间栏内部通过 `margin` 避开覆盖**
   - 在中间栏内部，为 `.inner` 设置 `margin: 0 200px;`（左右外边距等于左右栏宽度）。
   - 这样 `.inner` 的内容区域就被左右挤压缩小，正好位于左右栏之间的空白处，而不会被覆盖的左右栏遮挡。

```html
<div class="container">
  <div class="middle">
    <div class="inner">中间自适应</div>
  </div>
  <div class="left">左栏 200px</div>
  <div class="right">右栏 200px</div>
</div>
```

```css
.middle {
  float: left;
  width: 100%;
}
.inner {
  margin: 0 200px; /* 左右栏宽度 */
  background: lightgray;
}
.left {
  float: left;
  width: 200px;
  margin-left: -100%;
  background: lightblue;
}
.right {
  float: left;
  width: 200px;
  margin-left: -200px;
  background: lightgreen;
}
```
