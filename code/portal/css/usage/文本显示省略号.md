# 单行、多行文本显示省略号

## 实现

1. 单行显示文本，超出显示省略号（需要加宽度 `width` 属性来兼容部分浏览器）：

    ```css
    width:200px;
    overflow: hidden;
    text-overflow:ellipsis;
    white-space: nowrap; /* 让文本在一行内显示 */
    ```

2. 多行文本显示，限制行数，超出显示省略号，如下实现：

    ```css
    width:200px;
    display: -webkit-box;
    -webkit-box-orient: vertical;
    -webkit-line-clamp: 3;
    overflow: hidden;
    ```

## text-overflow 不会继承

`text-overflow` 属性不会被继承到子元素中，如果需要在子元素中也应用省略号效果，需要在子元素的 CSS 中重新设置 `text-overflow: ellipsis;`。

```html
<head>
  <style>
    .parent {
      width: 200px;
      overflow: hidden;
      text-overflow: ellipsis;
      white-space: nowrap;
    }
    .child {
      text-overflow: ellipsis; /* 需要在子元素中重新设置 */
    }
  </style>
</head>
<body>
  <div class="parent">
    这是一个很长的文本，应该显示省略号。
    <div class="child">这是子元素的文本，也应该显示省略号。</div>
  </div>
</body>
```
