# 安装

## 安装 Erlang

- 到 [官网](https://www.erlang.org/downloads) 下载 windows installer。
- 运行安装程序
- 添加环境变量。将 `C:\Program Files\Erlang OTP\bin` 添加到 `Path` 变量
- 在命令行输入 `erl -version` 看是否安装成功

## 安装 RabbitMQ

`C:\Program Files\RabbitMQ Server\rabbitmq_server-4.0.7\sbin`

- 到 [官网的 windows 安装页面](https://www.rabbitmq.com/docs/install-windows)，看到 `Using the Installer`，在 `Direct Downloads` 那里直接下载安装包
- 运行安装程序
- 添加环境变量。将 `C:\Program Files\RabbitMQ Server\rabbitmq_server-4.0.7\sbin` 添加到 `Path` 变量
- 执行命令**启用管理插件，可以使用 Web 管理界面**
  - `rabbitmq-plugins enable rabbitmq_management`
- 插件启动后，执行命令重启 RabbitMQ 服务
  - `rabbitmq-service restart`
  - 或者在 windows 开始菜单运行 `RabbitMQService - remove`，然后再运行 `RabbitMQService - start`
- 访问 `http://localhost:15672/#/` ，用户名和密码都为 `guest` ，登录
- 测试
  - 在 `Queues and Streams` 里面点击 `Add a new queue`
  - 在 `name` 输入名称，点击 `Add queue` 创建队列
  - 在 `All queues` 里面找到刚刚创建的队列，点击名称
  - 点击 `Publish message` ，在 `Payload` 输入 `Hello, World`。点击 `Publish message` 发送信息
  - 点击 `Get messages` ，点击 `Get Message(s)` ，看是否获取到刚刚发送的消息
  - 可以在 `Overview` 看到队列的概览

`RabbitMQ` 服务的默认端口为 `5672`
