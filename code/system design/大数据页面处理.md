# 大数据页面处理

在前端开发中，处理大数据量的页面时，常常会遇到性能瓶颈。为了提升用户体验，可以采用以下几种技术手段：

- 虚拟列表（Virtual Scrolling）
- 图片懒加载（Lazy Loading）
- 分页加载（Pagination）
- 数据分片（Data Chunking）

### 虚拟列表和图片懒加载组合使用

✅ 优先考虑图片懒加载

- 图片数量 < 500张
- 每张图片都有单独的DOM节点
- 主要问题是网络请求过多

✅ 考虑虚拟列表 + 懒加载

- 图片数量 > 1000张
- 图片以列表/网格形式排列
- 同时存在DOM节点过多和请求过多的问题
