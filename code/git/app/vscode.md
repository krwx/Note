# vscode

## 提交部分修改的文件

1. 在 Source Control 的 Changes 视图中，点击 Changes 文件夹的要提交的文件旁边的加号图标（+），将其暂存，放到 Staged Changes 文件夹中。
2. 填写 commit message，点击 commit 按钮提交。
3. pull 代码
4. 点击 Graph 视图的 push 按钮提交代码

## merge branch

1. **打开 Git 面板**：点击左侧活动栏的“源代码管理”图标（或按 `Ctrl+Shift+G` / `Cmd+Shift+G`）。

2. **触发合并**：点击面板顶部的 `...`（更多操作）菜单，在下拉列表中选择 `Branch` -> `Merge Branch...`。

3. **选择分支**：从弹出的列表中选择你要合并的源分支，VS Code 就会自动执行合并操作。

4. **推送合并后的代码**：点击面板顶部的 `...`（更多操作）菜单，在下拉列表中选择 `Push`，将合并后的代码推送到远程仓库。
