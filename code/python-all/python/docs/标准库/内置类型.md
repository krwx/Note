# 内置类型

## 迭代器类型

`Python` 支持在容器中进行迭代的概念。 这是通过使用两个单独方法来实现的；它们被用于允许用户自定义类对迭代的支持。

容器对象要提供 `iterable` 支持，必须定义一个方法:

- `container.__iter__()`
  - 返回一个 `iterator` 对象。
  - 该对象需要支持下文所述的**迭代器协议**。

迭代器对象自身需要支持以下两个方法，它们共同组成了**迭代器协议**:

- `iterator.__iter__()`
  - 返回 `iterator` 对象本身。 这是同时允许容器和迭代器配合 `for` 和 `in` 语句使用所必须的。

- `iterator.__next__()`
  - `iterator` 中**返回下一项**。
  - 如果已经没有可返回的项，则会引发 StopIteration 异常。
