# 字典

## 函数

### items()

`items()` 方法以列表**返回视图对象**，是一个可遍历的 `key/value` 对。

`dict.keys()、dict.values() 和 dict.items()` 返回的都是视图对象（ view objects），提供了字典实体的动态视图，这就意味着**字典改变，视图也会跟着变化**。

视图对象不是列表，不支持索引，可以使用 `list()` 来转换为列表。

我们不能对视图对象进行任何的修改，因为**字典的视图对象都是只读的**。

```py
tinydict = {'Name': 'Runoob', 'Age': 7}
print ("Value : %s" %  tinydict.items())

# 结果
# Value : dict_items([('Age', 7), ('Name', 'Runoob')])
```

遍历：

```py
d = {'Name': 'Runoob', 'Age': 7}
for i,j in d.items():
    print(i, ":\t", j)

""" 
输出：
Name :   Runoob
Age :    7
"""
```

### get()

语法：`get(key, default=None)`

如果 key 存在于字典中则返回 key 的值，否则返回 default。 如果 default 未给出则默认为 None，因而此方法绝不会引发 KeyError。

```py
tinydict = {'Name': 'Runoob', 'Age': 7}

print ("Age : %s" %  tinydict.get('Age')) # 返回 7
print ("Sex : %s" %  tinydict.get('Sex', "NA")) # 返回 NA
print ("Weight : %s" %  tinydict.get('Weight')) # 返回 None
```

### pop()

语法：

```py
pop(key)
pop(key, default)
```

如果 key 存在于字典中则将其移除并返回其值，否则返回 default。  
如果 default 未给出且 key 不存在于字典中，则会引发 KeyError。

```py
tinydict = {'Name': 'Runoob', 'Age': 7, 'Class': 'First'}

print (tinydict.pop('Name')) # 删除 'Name' 键，并返回对应的值 Runoob
print (tinydict) # {'Age': 7, 'Class': 'First'}
print (tinydict.pop('Sex', "NA")) # 'Sex' 键不存在，返回默认值 NA
print (tinydict.pop('Weight')) # 'Weight' 键不存在，抛出 KeyError
```
