# 文件路径过长问题

在 Windows 系统中，文件路径长度超过 260 个字符时会出现问题。

例如，写入一个文件路径为 `C:\Users\someUser\someDirectory\Documents\Note\temp.md` 的文件时，如果路径长度超过 260 个字符，就会出现错误。

```py
def some_write_file(file_path):
    with open(file_path, 'w') as f:
        f.write("Some content")
# 会报 FileNotFoundError: [Errno 2] No such file or directory: 'C:\\Users\\someUser\\someDirectory\\Documents\\Note\\temp.md'
```

解决方法：

1. 使用短一点的文件路径。
2. 在 Windows 10 中，可以启用长路径支持，方法是在注册表中设置 `LongPathsEnabled` 键值为 1。
