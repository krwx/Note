## Cell

属性：

- column_letter
- row
- column
- value
- coordinate. example: "A2"

### offset()

语法：`offset(row=0, column=0)`

Returns a cell location relative to this cell.

## Font

- strike: bool 类型。中划线
- color
- bold
- underline
- outline
- size

```py
# 判断单元格字体是否有中划线
if sheet.cell(row, column).font.strike:
  print("cell have strike")
```
