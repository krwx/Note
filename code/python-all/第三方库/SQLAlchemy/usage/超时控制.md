## 查询执行超时

SQLAlchemy提供了`execution_options`方法，可以设置查询的执行超时时间。例如：

```python
from sqlalchemy import text

# 以 seconds 为单位设置查询超时
with engine.connect() as conn:
    result = conn.execute(
        text("SELECT * FROM large_table WHERE complex_condition"),
        execution_options={"timeout": 30}  # 30秒查询超时
    )
    data = result.fetchall()
```

## 连接超时

连接超时是指在尝试建立数据库连接时，如果在指定的时间内无法成功连接，则会抛出超时异常。SQLAlchemy允许通过`create_engine`函数的参数来设置连接超时。例如：

```python
from sqlalchemy import create_engine

engine = create_engine(
    connection_string,
    connect_args={
        "timeout": 10,  # 连接/登录超时，单位秒，传给 pyodbc.connect()
    },
    pool_timeout=30,   # 从 SQLAlchemy 连接池取连接的等待时间，单位秒
    pool_pre_ping=True # 取连接前先检查连接是否还活着
)
```

## DBAPI 超时

可以通过监听连接事件来设置 DBAPI 超时。例如：

```py
from sqlalchemy import create_engine, event

engine = create_engine(
    connection_string,
    connect_args={"timeout": 10},
    pool_timeout=30,
    pool_pre_ping=True
)

@event.listens_for(engine, "connect")
def set_query_timeout(dbapi_connection, connection_record):
    dbapi_connection.timeout = 60  # SQL 执行超时，单位秒。SELECT, UPDATE, DELETE 等语句都会受到这个超时的限制
```

### 分别给 SELECT、UPDATE、DELETE 设置不同的超时

> 用不了 `cursor` 的方案

可以用两个 engine 分别设置不同的超时，然后在执行不同类型的 SQL 时使用对应的 engine：

```py
# 创建两个 engine，分别设置不同的超时
read_engine = create_engine(
    connection_string,
    connect_args={"timeout": 10},
    pool_timeout=30,
    pool_pre_ping=True,
)

write_engine = create_engine(
    connection_string,
    connect_args={"timeout": 10},
    pool_timeout=30,
    pool_pre_ping=True,
)

@event.listens_for(read_engine, "connect")
def set_read_timeout(dbapi_connection, connection_record):
    dbapi_connection.timeout = 180

@event.listens_for(write_engine, "connect")
def set_write_timeout(dbapi_connection, connection_record):
    dbapi_connection.timeout = 60
```

查询用：

```py
with Session(read_engine) as session:
    result = session.query(ReportList).filter_by(active="Y").count()
```

写入用：

```py
with Session(write_engine) as session:
    session.execute(update_stmt)
    session.commit()
```
