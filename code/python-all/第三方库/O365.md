`O365` 库是通过 Python 发送 Office 365 邮件的现代方案。它使用 OAuth 2.0 认证，比传统 SMTP 方式更安全可靠，尤其适合自动化任务。

## 发送邮件步骤

### 1. 安装

```bash
pip install O365
```

### 2. 配置应用

在使用`O365`库之前，需要在 **Microsoft Entra 管理中心** 注册一个应用程序来获取`client_id`和`client_secret`：

- **访问地址**：进入 [Microsoft Entra 管理中心](https://entra.microsoft.com/)，依次进入 **“应用注册”** > **“新注册”**。
- **填写信息**：为应用起个名字，在“重定向 URI”中选择“公共客户端/本机 (Mobile & Desktop)”，并填写`http://localhost`。
- **获取凭证**：注册成功后，保存“应用程序(客户端) ID”（即 `client_id`）。再进入 **“证书和密码”** > **“新客户端密码”** 生成一个 `client_secret` 并保存。
- **配置权限**：进入 **“API 权限”** > **“添加权限”** > **“Microsoft Graph”** > **“委托的权限”**。搜索并勾选 **`Mail.Send`**，最后点击“为[租户名]授予管理员同意”。

### 3. 认证账户

配置好应用后，在代码中初始化账户：

```python
from O365 import Account

# 你的凭证
credentials = ('your_client_id', 'your_client_secret')

# 初始化账户对象
# auth_flow_type='credentials' 表示使用客户端凭证模式，必须指定 tenant_id
account = Account(credentials, auth_flow_type='credentials', tenant_id='your_tenant_id')

if account.authenticate():
    print("非交互式认证成功。")
    # 认证后发送邮件的代码完全相同...
else:
    print("认证失败。")
```

可以指定 token 存储路径，避免每次运行都需要认证，后续 SDK 会从这里读取或更新认证 token：

```python
account = Account(credentials, auth_flow_type='credentials', tenant_id='your_tenant_id', token_backend=FileSystemTokenBackend(token_path='.', token_filename='o365_token.txt'))
```

如果账户的密码更新了，需要更新 `client_secret`，并重新认证。不需要更新 token 文件，SDK 会自动刷新 token。

### 4. 发送邮件

认证成功后，就可以用简洁的代码发送邮件了：

```python
# 获取当前用户的邮箱账户对象
mailbox = account.mailbox("myMailbox@example.com")

# 创建一封新邮件
message = mailbox.new_message()

# 设置收件人、主题和正文
message.to.add('recipient@example.com')
message.subject = '来自Python的问候'
message.body = '你好，这是一封由Python O365库发送的测试邮件。'

# 发送！
message.send()
print("邮件发送成功!")
```

## 进阶功能

### 添加附件

附件支持本地文件和网络文件。

```python
# 在 send() 方法之前添加附件
# 添加本地文件
message.attachments.add('path/to/your/file.pdf')
# 从 URL 添加文件
message.attachments.add('https://www.example.com/image.png')
```

### 发送 HTML 邮件

支持发送格式化的 HTML 邮件，提升阅读体验。

```python
html_content = """
<h2>这是一封HTML邮件</h2>
<p>内容可以是<strong>丰富</strong>的HTML格式。</p>
<ul>
    <li>项目1</li>
    <li>项目2</li>
</ul>
"""
message.body = html_content
message.body_type = 'html'  # 关键：将邮件内容设为HTML格式
```

### 发送给多人

可以同时发送给多个收件人，并可设置抄送（CC）、密送（BCC）。

```python
# 添加多个收件人
message.to.add(['user1@example.com', 'user2@example.com'])

# 添加抄送收件人
message.cc.add('cc@example.com')

# 添加密送收件人
message.bcc.add('bcc@example.com')
```
