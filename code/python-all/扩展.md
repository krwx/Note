### 设置参数

这里以 disable 规则为例：

1. Settings 界面
   1. 在 `Pylint: Args` 那里添加参数，例如：`--disable=C0116`
   2. 一个参数对应一行
2. 在 `settings.json` 添加。注意 `pylint.args` 是数组

    ```json
     "pylint.args": [
         "--disable=C0116",
         "--disable=C0114"
     ]
    ```
