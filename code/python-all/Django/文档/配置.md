## 核心

### FORCE_SCRIPT_NAME

默认： None

- 如果不是`“None”`，则这将用作任何 `HTTP` 请求中`“SCRIPT_NAME”`环境变量的值。  
- 此设置可用于覆盖服务器提供的`“SCRIPT_NAME”`值，该值可能是首选值的重写版本或根本不提供。  
- 它还被 `django.setup()` 用于在**请求/响应周期之外**设置 `URL` 解析器**脚本前缀**（例如在管理命令和独立脚本中），以便在提供`“FORCE_SCRIPT_NAME”`时生成正确的 `URL` 。

简单来说：用于**设置根 `URL` 路径前缀**
