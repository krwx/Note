# tip

## 时间

- 输入日期：`ctrl + ;`。格式：`yyyy/m/d`
- 输入时间： `ctrl + shift + ;`。格式：`hh:mm`

## other

锁定： view -> FreezePanes

列过滤：选中列，点击 Home -> sort & filer -> filter

列名从字母变成数字： `Options` -> `Formulas` -> 勾选 `R1C1 reference style` 。 `R1C1` 代表 `Row` 为 1 ， `Column` 为 1 .

取消隐藏 sheet ： Home -> Format -> Hide & Unhide -> Unhide sheet

单元格内的内容换行：`alt + enter`

隐藏 excel 文件：View -> Hide

取消隐藏 excel 文件：View -> Unhide，选择对应的 excel 文件，点击确定

添加按钮：Developer -> Insert -> Form Controls -> Button

自动适配列宽度：选中要操作的列 -> Home -> Format -> AutoFit Column Width
